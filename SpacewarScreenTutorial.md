# Mouse gestures #

| Gesture | Effect |
|:--------|:-------|
| Double click on an object | Select all units with the same exact type (i.e., all Fighter 1s) |
| Triple click on an object | Select all units with the same category (i.e., all Fighters) |
| Right click an object in the selection panel | Remove the particular unit from selection. |
| Double click on an object in the selection panel | Select all units with the same exact type |
| Triple click on an object in the selection panel | Select all units with the same category |
| CTRL + Double click an object in the selection panel | Remove units from selection with the same exact type |
| CTRL + Triple click an object in the selection panel | Remove units from selection with the same category |
